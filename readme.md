# is-github-url

> Get github URL (https or ssh) based on passed arguments

## Install

```
$ npm install is-github-url --save
```

## Usage
```js
var isGithubUrl = require('is-github-url');
```

## API

### `isGithubUrl(options)`
Get github URL (https or ssh) based on passed arguments

#### Params
- `url`
- `protocol`
